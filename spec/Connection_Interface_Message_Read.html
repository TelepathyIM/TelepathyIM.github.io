Objects implementing this interface must also implement:
  • org.freedesktop.Telepathy.Connection

Description

An interface on connections to mark a Channel messages as read or unread.

Methods

(Permalink)

MarkRead (a{sv}: ChannelRequest, s: MessageToken) → nothing

Parameters

  • ChannelRequest — a{sv} (Channel_Class)

    • A dictionary containing desirable properties, with the same semantics as the corresponding parameter to CreateChannel.

  • MessageToken — s (Protocol_Message_Token)

    • An identifier of the last read message. All messages up to this one should be marked as read. MUST NOT be an empty string.

Mark the referenced Channel history as read.

Possible Errors

  • Invalid Argument
    • Raised when one of the provided arguments is invalid.
(Permalink)

MarkUnread (a{sv}: ChannelRequest) → nothing

Parameters

  • ChannelRequest — a{sv} (Channel_Class)

    • A dictionary containing desirable properties, with the same semantics as the corresponding parameter to CreateChannel.

Manually mark a Channel as unread.

Possible Errors

  • Invalid Argument
    • Raised when one of the provided arguments is invalid.

Properties

Accessed using the org.freedesktop.DBus.Properties interface.

Read only

Types

Flags (Permalink)

Read_Reporting_Support_Flags — u

Flags indicating the level of support for read reporting on this connection, as found on the ReadReportingSupport property.
  • Mark_Unread (1)
    • Clients MAY mark Channels as unread.
